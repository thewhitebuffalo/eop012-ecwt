Source: scoped_plant_ecwt_release_20260625T161629Z · 15,947 non-Alaska EIA-860 plants · ECWT = lowest 0.2-percentile DJF hourly temperature (NERC EOP-012)
Analytical / provisional — not compliance-grade. Many plants are matched to a representative station far away and coverage is measured on a relaxed denominator, so values (e.g. California reading warm) can be unrepresentative. Use for exploration, not as an EOP-012 filing input. See the "Data-quality reality check" panel below.

Data-quality reality check — station representativeness

How far is each plant's assigned primary weather station? Only ~3% are within 25 km; most are 100 km+ away — the core reason these ECWT values are provisional.
Generated locally from the public repo thewhitebuffalo/eop012-ecwt. ECWT below 32°F is the EOP-012 applicability trigger. Known caveats are tracked as GitHub issues #1–#8 (station distance, relaxed coverage denominator, value quantization, missing caveats on the export). This dashboard is a self-contained local file (EOP012_ECWT_dashboard.html + data.js); it uses no internet connection.
