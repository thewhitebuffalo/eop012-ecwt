# ADR 0006: Exclude Marine Platforms From Land-Plant ECWT Composites

## Status

Accepted

## Date

2026-07-02

## Context

Issue #36 identified a confirmed data error in the
`scoped_plant_ecwt_adr0004_release_20260626T235840Z` release: 24 Maryland
plants and Pickford Solar in Michigan received implausibly cold published ECWTs
because their entire 0.2-percentile cold tail was filled from marine platforms.

The Maryland cluster was driven by station `997994-99999`, a Chesapeake Bay
marine platform. Its FM-13 ship/marine observations were pinned at roughly
-37 F to -39.8 F from 2019-12-27 through 2020-01-24, a mild period when nearby
land observations in the Washington, DC / southern Maryland region were in the
30s to 50s F. Pickford Solar was similarly affected by Great Lakes platform
`998203-99999`.

Two properties make this a methodology issue, not just a one-off bad row:

1. Marine platforms measure water-modulated air temperature. Even when healthy,
   those observations are not representative of land generating-site
   conditions for the national land-plant ECWT layer.
2. Marine sensor failures can rail near -40 F while still passing NOAA/NCEI
   global-plausibility quality checks. Quality flags alone are therefore not a
   sufficient defense against this failure mode.
3. ADR-0004 requires observed station data and per-hour provenance, but it did
   not explicitly restrict the automated observational hierarchy to land
   stations for land plants. The current release proved that omission can alter
   published ECWT values materially.

The confirmed issue was detected from the release CSV and cold-tail provenance:
plants more than 10 F below their state median whose cold-tail provenance was
dominated by `997xxx`, `998xxx`, or `999xxx-99999` marine-style station IDs.

## Decision

For land generating plants, the automated ECWT station hierarchy is restricted
to land stations.

Marine and ship platforms are excluded from:

- automated station candidate selection for land plants;
- composite-hour fallback fill for land plants;
- any published land-plant ECWT result generated by this pipeline.

The exclusion covers NOAA/ISD marine platform evidence including FM-13
ship/marine reports and buoy/C-MAN style station identifiers in the
`997xxx`, `998xxx`, and `999xxx-99999` ranges. The implementation may use both
station metadata and deterministic station-id predicates, because the current
release data demonstrates that the problematic platforms are identifiable from
the station IDs present in provenance.

This ADR does not add a general low-temperature climatology override, regional
outlier trimming rule, or manual value substitution. It only removes marine
platforms from the land-plant source hierarchy. Land plants must be recomputed
from the remaining representative land-station observations and documented
composites.

## Implementation Rules

1. Add a reusable marine-platform exclusion predicate and apply it wherever
   automated land-plant station candidates or fallback towers are chosen.
2. Existing releases remain traceable under their original methodology version.
   New releases after this decision must use a new methodology version and must
   cite ADR-0006 in the run metadata, dashboard/release notes, and RSAW source
   documentation where relevant.
3. Recompute the current release for every plant whose current composite or
   cold-tail provenance references an excluded marine platform. The known
   corrupted set is 25 plants, but the rerun scope must be enumerated from the
   current release before calculation rather than assumed from the issue
   examples.
4. The regenerated release must pass the validator and must publish a
   data-only SHA-256 manifest. Dashboard hashes remain out of data manifests.
5. Add release-level validation guardrails so a future release cannot silently
   publish this failure class again.

## Consequences

- The 24 Maryland plants and Pickford Solar must receive corrected ECWT values
  based on land-station composites. The expected Maryland result cluster should
  return near the state distribution rather than the failed marine-platform
  -23 F values.
- A marine observation can still appear in raw source tables for auditability,
  but it is not eligible for automated land-plant ECWT publication.
- This narrows ADR-0004's observational hierarchy: observed data remain
  required, but not every observed NOAA/ISD platform is representative for a
  land generating site.
- The issue-specific remedy is documented for RSAW R1 "sources of temperature
  data / adjustments" responses: the adjustment is exclusion of non-representative
  marine platforms, not a post hoc edit to the ECWT value.
